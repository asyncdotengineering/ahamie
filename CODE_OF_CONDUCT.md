# Contributor Covenant Code of Conduct

We adopt the [Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) verbatim.

## Reporting

Report violations to **conduct@ahamie.dev**. Reports are read by the maintainer council only. We will respond within 72 hours.

## Enforcement

The council follows the Contributor Covenant Enforcement Ladder: correction → warning → temporary ban → permanent ban. Enforcement decisions are appealable in public.
