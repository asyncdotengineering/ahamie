# @ahamie/docs

The Ahamie documentation site (Astro + Starlight, Diataxis-discipline).

```bash
pnpm install
pnpm dev      # http://localhost:4321
pnpm build
```

Diataxis quadrants:

- **Tutorials** (learning-oriented): first-agent, closed-loop, software-factory
- **How-to guides** (problem-oriented): connect-slack, add-connector, hidden-golden, migrate-db, multi-tenant
- **Reference** (information-oriented): packages, config, cli, schema, trigger-dsl, proxy-invariants, eval-metrics
- **Explanation** (understanding-oriented): closed-loop, wrap-mastra, sensor-isolation, tenancy-ladder, naming
