# @ahamie/ui

Five v0 primitives in shadcn registry format (MIT-licensed):

| Primitive | Purpose |
|---|---|
| `AgentRunTree`   | Tree view of agent steps for one run |
| `RunConsole`     | SSE-fed run console |
| `ApprovalInbox`  | Inbox of suspended runs awaiting humans |
| `ConnectorSetup` | OAuth scope-grant form |
| `ManifestEditor` | JSON manifest editor |

```bash
pnpm exec ahamie ui add agent-run-tree run-console approval-inbox
```
